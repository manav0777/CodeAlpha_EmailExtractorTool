# 📧 Email Extractor Tool

A modern Flask-based Email Extractor Tool supporting TXT, CSV, XLSX, and PDF files.

---

## 🚀 Features

- Extract emails from:
  - TXT files
  - CSV files
  - XLSX files
  - PDF files

- Modern Dark UI
- Copy extracted emails
- Remove duplicate emails
- Clear output button
- Loading animation
- Responsive design

---

## 🛠️ Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- Pandas
- PyPDF2
